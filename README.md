<pre>
  
  Зачем этот микропресет?

</pre>

— Чтобы модели стали более творческими, помнящими и умными.  
  Меньше забывали, повторялись и юзали заезженности по сравнению с длинными пресетами

— Сильно экономит токены, позволяя ключам жить дольше

— Супер-органичная игра на русском  
  с возможностью английского (по языку ваших постов)

---

<pre>

  Для каких моделей?

</pre>

Считаю универсалом, тестился на:

- **Gemini 2.5–3 Pro** — лучший результат  
  фиксит ценз, повторюшничество и клише

- **Claude 4.1–4.5 Sonnet / Opus** — юзабельно  
  спасают memory book и summaries

- **GPT 5.1+** — юзабельно  
  при агрессивной цензе включайте думалку  
  или используйте синонимы

<pre>
  
  Что интересного в пресете?

</pre>

### Антиценз думалка

— Пресет можно использовать в двух режимах:  
  без думалки и с думалкой (2 модуля)

— Думалка эффективнее регексов, брейнфаков  
  и длинных NSFW-промтов

— Встроен модуль ограничения длины ~250 слов  
  (если с думалкой пишет простыни)

— Gemini иногда игнорирует кастомную думалку  
  это нормально, просто рерольте

<pre>
  
  Какие модули есть ещё?

</pre>

- **Котячий CSS** — милый интерфейс с русскими кнопками и котиками  
  выключен по умолчанию  
  ⚠ использовать только с HTML-регексом от Селии  

- **Киберпанк CSS** — консольный интерфейс с кнопками (eng)

- **Микрокот + OOC** — соролевик и комментатор  
  милый зумер с кошачьими повадками  
  обращаться через OOC, может слегка фиксить ценз

- **Русский язык** — фикс, если боты отвечают на английском  
  лайфхак: дублируйте зарубежные имена на русском в карточке

- **Формат письма** — " " вместо « » и —  
  (спасибо @xvcrrd)

- **Антиэхо Gemini** — снижает эхо  
  особенно при включённой думалке

- **Антивсезнание** — персонажи не знают лишнего

- **Не пиши за юзера + мини антиэхо**

- **Пиши за юзера** — фанфик-режим

- **3 выбора** — варианты развития сюжета в конце поста

- **Сочный стиль** — меньше клише, больше выразительности

- **Драма-квин тон**

- **Литературный тон**

- **Слоубёрн / быстрое развитие**

- **Реалистик хорни**

- **1 и 3 лицо повествования**

- **Стиль чата** — короткие сообщения как в мессенджере
